### Github workshop notes
```bash
git config --global user.email "email@example.com" #email setup
git config --global user.name "Andres Vargas" #username Setup
git clone https://github.com/username/reponame.git #Copy repo
git log #show history
git status 
git add HelloWorld.cpp #add whole file
git add --patch #add changes selectively
git commit #create commit
git diff #see diff
git checkout --patch #discard changes selectively
git push #submit changes to remote
git reset --hard abcdefg #reset workspace to commit id abcdefg
git pull #Sync workspace with remote
git checkout -b "DevBranch" #new branch
git branch -l #branch list
git checkout master # move between branches
git checkout DevBranch # move between branches
git commit --amend # change last commit description
git push origin DevBranch #submit changes to remote branch
```
